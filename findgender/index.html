Gender Identification

This is a demontration server to validate the gender, given the first name

How is it done?

Gender is identified using Machine Learning. ML package in R.

This engine has an in-built database of a million records taken from publicly available sources and the first level of prediction is based on that. In case a prediction has not been made possible, or the match probability is very low, the engine calls an external API for aid. In extremely rare cases where both methods fail, the engine displays and empty string.

Please enter only the first name of the person whose gender you want to identify. The gender and probability match of the resolution will be displayed on the click of the button along with the method that has been adopted.

Limitations ?

Currently, American and English names are resolved through this engine. Due to the cosmopolitan nature of the North American demographics, a good majority of the names get resolved. You may try this out using the panel on the right side.
